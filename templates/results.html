Installation Instructions

{{ out_name }} is a wrapper that contains two separate mod ZIPs. You must install both as separate mods — the game needs each one active at the same time.

  1. Unzip {{ out_name }} — you'll find two files inside:
    • {{ out_name | replace('.zip', '') }}-data.zip — modified game data (DAT, tech trees)
    • {{ out_name | replace('.zip', '') }}-ui.zip — civ names, icons, AI stubs
  2. In AoE2:DE, open Mods → My Mods and click Import Mod to install each zip as its own mod.
  3. Make sure both mods show a green checkmark (active) before launching a game.
If only the data mod is active, your civs will play correctly but show their old vanilla names and icons in the selection screen. Both mods must be active together.
