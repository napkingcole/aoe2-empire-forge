How It Works

A quick guide to building your first custom civilization mod.

What is AOE2: Empire Forge?

AOE2: Empire Forge converts civilization designs made in KrakenMeister's Civ Builder into fully playable Age of Empires II: Definitive Edition mods — no manual file editing required.

KrakenMeister's builder lets you design civs in your browser (tech tree, unique unit, bonuses, etc.) and export a .json file per civ. Empire Forge reads those JSON files, patches them into the game's internal data (empires2_x2_p1.dat), and packages everything into a mod you can drop straight into AoE2:DE.

What You Need
  • One or more KrakenMeister civ JSON files — design your civs at krakenmeister.com/civbuilder, then use the Export JSON button on each civ's page.
  • Your AoE2:DE DAT file — empires2_x2_p1.dat, found inside your game installation under resources/_common/dat/. Empire Forge will try to find it automatically. (Steam: …/Steam/steamapps/common/AoE2DE/resources/_common/dat/; Microsoft Store: C:\Program Files\WindowsApps\Microsoft.MSPhoenix_[version]\resources\_common\dat\)
  • That's it. No Python knowledge, no modding experience, no manual file editing needed.
The Three Steps
1
Upload

Drop your KrakenMeister civ JSON files onto the upload page and confirm the path to your DAT file. You can upload as many civs as you like in one go — there's no hard limit, though AoE2:DE has a finite number of available civ slots to replace.

2
Configure

Drag your civs into the order you want them to appear in-game, and choose which vanilla civ slot each one replaces. Any vanilla civ can be a replacement target — the original will be overwritten in the mod (your actual game files are never touched).

3
Build & Download

Click Build Mod. Empire Forge processes each civ — applying tech trees, bonuses, unique units, and unique techs — then packages the result into a .zip you can download and install directly into AoE2:DE's mod folder.

Installing the Mod

The exe will output a zip file that contains two separate mod ZIPs. You must install both as separate mods — the game needs each one active at the same time.

  1. Unzip your mod download and you'll find two files inside:
    • yourmodname-data.zip — modified game data (DAT, tech trees)
    • yourmodname-ui.zip — civ names, icons, AI stubs
  2. Unzip both of these to your local mods folder - usually located in C:\Users\yourname\Games\Age of Empires 2 DE\12345678\mods\local, where '12345678' is the folder for profile. You may have to test.
    Pro Tip: Create a shortcut for this folder and pin it to your File Explorer sidebar
  3. Make sure both mods show a green checkmark (active) before launching a game.
If only the data mod is active, your civs will play correctly but show their old vanilla names and icons in the selection screen. Both mods must be active together.
Limitations

Most KrakenMeister bonuses are fully supported, but a small number aren't implemented. See the Known Limitations page for a complete list of what's skipped and why.

If a bonus your civ uses isn't supported, it's silently skipped — the rest of the civ builds normally. The Results page will tell you exactly which bonuses were applied and which were skipped.
